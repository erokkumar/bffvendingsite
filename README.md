## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/erokkumar/bffvendingsite.git
   ```

2. Navigate to the project directory:
   ```bash
   cd BFFVending.in-main
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

## Usage

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
